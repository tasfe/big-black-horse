第一，先确定您使用的是否是古老的IE6浏览器？可以通过以下地址来检测：http://jankerli.sinaapp.com/detect-browsers/。
如果有看到“版本：IE 6.0”的话，请赶紧升级您的浏览器吧，因为淘宝的装修平台在利用“编辑元素”这一功能来装修店铺的时候，会出现某些模块错位的现象，请更新您的浏览器吧！IE8浏览器下载地址：
如果没有的话，证明您现在使用的是比较高级先进的浏览器了，请看第二步；

第二，确定您是否是直接在编辑器里修改图片、文字、链接等神马的？
如果是的话，请使用编辑器提供的“编辑元素”，因为我们的模板把一些需要您手动修改的地方都通过此方式展现出来了，以减少您触碰代码的烦恼；
如果否的话，请看第三步；

第三，确定您使用的编辑器是新版还是旧版的？
如果您使用的是新版的编辑器，请点击编辑器右上角的“切换”按钮，把您的编辑器切换成旧版的吧，因为新版的编辑器还不稳定，在装修时也会出现模块错位的现象。
